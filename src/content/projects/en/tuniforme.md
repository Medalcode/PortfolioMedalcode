# Tuniforme

Academic project simulating an online store ("Fake Store") specializing in school uniforms.

## Key Features

- **Dynamic Catalog**: Real-time product filtering and search.
- **Persistent Cart**: Simulated purchase state management.
- **Responsive Design**: Adaptable to mobile and desktop with Bootstrap.

## What makes it special?

It is a complete simulation of a **Full Stack** environment. It not only showcases products but also implements real e-commerce business logic, from navigation to checkout, all deployed on modern serverless infrastructure (**Google Cloud Run**).

## Context

This project was developed to consolidate knowledge in frontend and backend web development, as well as cloud deployment practices.
