# Myna: Intelligent Data Mining Tool

Myna is a powerful, modular data mining application designed to perform descriptive statistics, data cleaning, outlier detection, clustering, and interactive visualization on CSV and Excel datasets.

## Key Features

- **Refactored V5.0**: Now powered by FastAPI, Hexagonal Architecture, and Vanilla JS.
- **Data Cleaning**: Tools for handling missing values and inconsistencies.
- **Advanced Analysis**: Outlier detection and clustering capabilities.
- **Interactive Visualization**: Explore your data visually.

## Technologies

Built with **FastAPI** (Python) for the backend and **Vanilla JS** for the frontend, following a clean **Hexagonal Architecture**.
