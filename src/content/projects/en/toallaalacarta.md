# Toallaalacarta

A **Storeplate** implementation: High-performance e-commerce starter and boilerplate built with Astro, Shopify, and Tailwind CSS.

## Key Features

- **Storefront API Integration**: Fetches dynamic products directly from Shopify.
- **Optimized Performance**: 99% PageSpeed score with static generation and partial hydration.
- **Modern Stack**: Built with Astro 5, TailwindCSS v4, and TypeScript.
- **Cart & Checkout**: Full shopping experience integration.
- **SEO & I18n**: Optimized for search engines and ready for global reach.

## Technologies

This project demonstrates the power of **Island Architecture** (Astro) combined with headless e-commerce (Shopify) for blazing fast online stores.
