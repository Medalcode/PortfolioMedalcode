# Kitsune API Template

An enterprise-grade base template for the rapid development of robust APIs using **FastAPI**.

## Features

- **Modern FastAPI**: Leverages the latest in Python web development.
- **Async SQLAlchemy**: Optimized and modern database interaction.
- **Comprehensive Security**: JWT OAuth2 authentication, Bcrypt password hashing.
- **Deployment Ready**: Docker configuration and CORS pre-configured.

## What makes it special?

Provides a clean and scalable architecture from day one, saving weeks of initial setup and ensuring security and performance best practices.
