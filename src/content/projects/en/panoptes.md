# Panoptes Dashboard

A centralized command center for **Autonomous Job Search**, leveraging AI agents to find, verify, and apply for opportunities automatically.

## Key Features

- **Agent Orchestration**: visualizes the lifecycle of job search agents from Upload to Application.
- **Real-time Match Scoring**: Calculates compatibility percentages for every job listing found.
- **Multi-Source Aggregation**: Pulls data from sources like ChileTrabajos and others into a unified view.

## Technologies

Built with a robust **Python** backend for scraping and agent logic, paired with a modern **React/Next.js** frontend hosted on **Vercel** for monitoring active targets and application status.
