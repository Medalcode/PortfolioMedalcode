# Kitsune API Template

Plantilla base de nivel empresarial para el desarrollo rápido de APIs robustas utilizando **FastAPI**.

## Características

- **FastAPI Moderno**: Aprovecha lo último en desarrollo web con Python.
- **SQLAlchemy Async**: Interacción con bases de datos optimizada y moderna.
- **Seguridad Completa**: Autenticación JWT OAuth2, hashing de contraseñas con Bcrypt.
- **Listo para Desplegar**: Configuración Docker y CORS preconfigurada.

## ¿Qué lo hace especial?

Proporciona una arquitectura limpia y escalable desde el primer día, ahorrando semanas de configuración inicial y garantizando las mejores prácticas de seguridad y rendimiento.
