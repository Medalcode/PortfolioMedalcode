# Panoptes Dashboard

Un centro de comando centralizado para la **Búsqueda Laboral Autónoma**, aprovechando agentes de IA para encontrar, verificar y postular a oportunidades automáticamente.

## Características Principales

- **Orquestación de Agentes**: Visualiza el ciclo de vida de los agentes de búsqueda, desde la carga hasta la postulación.
- **Puntuación de Coincidencia en Tiempo Real**: Calcula porcentajes de compatibilidad para cada oferta laboral encontrada.
- **Agregación Multi-Fuente**: Consolida datos de fuentes como ChileTrabajos y otros en una vista unificada.

## Tecnologías

Construido con un backend robusto en **Python** para scraping y lógica de agentes, junto con un frontend moderno en **React/Next.js** alojado en **Vercel** para monitorear objetivos activos y el estado de las postulaciones.
