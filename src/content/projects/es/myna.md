# Myna: Herramienta Inteligente de Minería de Datos

Myna es una potente aplicación modular de minería de datos diseñada para realizar estadística descriptiva, limpieza de datos, detección de valores atípicos, clustering y visualización interactiva en conjuntos de datos CSV y Excel.

## Características Clave

- **Refactorización V5.0**: Ahora impulsada por FastAPI, Arquitectura Hexagonal y Vanilla JS.
- **Limpieza de Datos**: Herramientas para manejar valores faltantes e inconsistencias.
- **Análisis Avanzado**: Capacidades de detección de valores atípicos y clustering.
- **Visualización Interactiva**: Explora tus datos visualmente.

## Tecnologías

Construido con **FastAPI** (Python) para el backend y **Vanilla JS** para el frontend, siguiendo una limpia **Arquitectura Hexagonal**.
