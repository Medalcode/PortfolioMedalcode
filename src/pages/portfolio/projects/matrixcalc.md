# MatrixCalc

MatrixCalc es una herramienta diseñada para estudiantes y profesionales que necesitan realizar cálculos matriciales de manera rápida y precisa.

## Funcionalidades

- **Operaciones Básicas**: Suma, resta y multiplicación de matrices.
- **Operaciones Avanzadas**: Cálculo de determinantes, inversas y transpuetas.
- **Interfaz Intuitiva**: Diseño fácil de usar para ingresar datos matriciales.

## Despliegue

La aplicación backend y frontend se encuentra desplegada en **Google Cloud Run**, asegurando escalabilidad y alta disponibilidad.
