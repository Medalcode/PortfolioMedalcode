# 🗺️ Project Roadmap

This document outlines the future development plans and upcoming sprints for the PortfolioMedalcode project.

## 🚀 Sprint 3 - Performance y SEO (Upcoming)

Focus: Optimizing application performance and search engine visibility.

- [ ] **Image Optimization**: Implement advanced image optimization strategies.
- [ ] **Self-host Fonts**: Host fonts locally to improve load times and privacy.
- [ ] **SEO Enhancements**:
  - [ ] Implement Schema.org structured data.
  - [ ] Optimize Open Graph (OG) images.
- [ ] **PWA Implementation**: Transform the site into a fully functional Progressive Web App.

## 🛠️ Sprint 4 - CI/CD & Automation

Focus: Automating development workflows and ensuring robustness.

- [ ] **GitHub Actions**: Create comprehensive CI/CD workflows.
- [ ] **Dependabot**: Configure automated dependency updates.
- [ ] **Husky**: Implement git hooks for pre-commit checks (linting, testing).
- [ ] **Error Tracking**: Integrate an error tracking solution (e.g., Sentry).

## 🔮 Future Ideas

- [ ] **Blog Comments**: Add a commenting system (e.g., Giscus).
- [ ] **Newsletter**: Integrate a newsletter subscription form.
- [ ] **CMS Integration**: Connect to a Headless CMS for easier content management.
